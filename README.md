# Система классификации писем

Автоматизированный конвейер для чтения, классификации и сортировки входящих писем по категориям.

## Запуск проекта

sh run.sh

## Структура проекта

* `app/main.py` - точка входа в программу
* `app/classifier.py` - логика распределения писем
* `app/organizer.py` - сортировка файлов по папкам категорий
* `app/stats.py` и `app/reporter.py` - статистика и вывод результатов
* `app/logger.py` - запись истории в process.log

## Запуск тестов

pytest tests/
